# http fetch

### inspiring links
- https://bun.sh/
- https://bun.sh/guides/http/fetch

### 1st time setup
- `cd ~/dev/repository/gitjs/my_bun/http_fetch`
- `bun init`

### run
- `cd ~/dev/repository/gitjs/my_bun/http_fetch`
- `pnpm i`
- `bun run get_ip.ts`
- `bun run get_bitcoin_price.ts`
- `bun run post_hello_world.ts`
